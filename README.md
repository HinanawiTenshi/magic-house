# Magic House

A CG Project by OpenGL

## Usage

### Prerequisites

- `GLFW3` Library

- `GLM` 0.9.8-5

- `Assimp` 4.1.0

### Makefile

You can use `make` to build and run.

```bash
$ git clone https://github.com/HinanawiTenshi/magic-house.git
$ cd magic-house
$ make
```

Other `make` commands are:

- `clean` clean

- `build` build

- `all` clean, build and run

- `rebuild` clean and build

## For Contributors
